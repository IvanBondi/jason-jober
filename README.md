# Jason Jober 🤖

AI-powered job monitoring agent for Upwork & Freelancer.

## Setup

### Environment Variables (Railway)
```
TELEGRAM_TOKEN=your_bot_token
CHAT_ID=your_chat_id
```

## How it works
1. Monitors RSS feeds every 5 minutes
2. Filters jobs by keywords and minimum budget ($20+)
3. Sends Telegram notifications for relevant jobs

## Categories monitored
- Article writing & blog posts
- Data entry & web research
- Excel / Google Sheets
- Python scripts & automation
- Translation
